# rust-file-squeezer

## Command to Compress a file 

Example :

cargo run compress <filename>

```bash
cargo run compress Cargo.toml
```

## Command to Decompress a file 

Example :

cargo run decompress <filename>

```bash
cargo run decompress Cargo.toml.squeeze 
```
